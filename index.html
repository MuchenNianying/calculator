工资薪金收入个税

工资、薪金所得，是指个人因任职或受雇而取得的工资、薪金、奖金、年终加薪、劳动分红、津贴、补贴以及与任职或受雇有关的其他所得。

工资个税预扣预缴计算公式

当月个税 = （累计收入 - 累计五险一金 - 累计专项附加扣除 - 累计减除费用）× 预扣税率 - 速算扣除数 - 累计已缴纳税额
应纳税所得额 = 累计税前工资收入 - 累计五险一金（个人缴纳部分） - 累计专项附加扣除 - 累计减除费用
累计应纳税额 = 应纳税所得额 × 税率 - 速算扣除数
当月应纳税额 = 累计应纳税额 - 累计已缴纳税额
累计减除费用 = 员工当年在职月数 × 5000

房贷计算功能说明

本工具支持等额本息和等额本金两种常见还款方式的计算，帮助用户预估每月还款金额、总利息支出以及详细的还款计划。计算结果仅供参考，实际还款金额以银行审批为准。

还款方式比较

等额本息：每月还款额固定，前期利息占比较高，本金占比较低；随着还款期限推进，利息占比逐渐降低，本金占比逐渐升高。这种方式适合收入稳定、希望每月还款压力均衡的购房者。

等额本金：每月还款本金固定，利息逐月递减，因此每月总还款额呈递减趋势。前期还款压力较大，但总利息支出较等额本息少，适合前期还款能力较强、希望减少总利息支出的购房者。

计算注意事项

  • 贷款总额：指实际贷款金额，单位为万元（如贷款100万则输入100）
  • 年利率：请输入银行实际执行的贷款利率，一般为基准利率上下浮动
  • 贷款年限：最长不超过30年，不同银行和贷款类型可能有不同限制
  • 计算结果仅供参考，实际还款金额可能因银行政策、还款方式调整等因素有所不同

BMI计算器

BMI计算功能说明

BMI（Body Mass Index，身体质量指数）是国际上常用的衡量人体胖瘦程度以及是否健康的一个标准。本工具通过身高和体重数据计算BMI值，并提供相应的健康评估建议。

计算公式

BMI = 体重(kg) ÷ 身高(m)的平方

BMI标准范围

  • 偏瘦：BMI < 18.5
  • 正常：18.5 ≤ BMI < 24
  • 超重：24 ≤ BMI < 28
  • 肥胖：BMI ≥ 28

注意事项

  • BMI是一个参考指标，不能完全代表健康状况
  • 运动员、孕妇、老年人等特殊人群可能不适用BMI标准
  • 如需更准确的健康评估，建议咨询专业医生
  • 保持健康的体重需要结合合理饮食和适量运动
