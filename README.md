# 多功能计算器

一个集成了多种实用计算工具的在线平台，提供便捷、高效的计算服务。

## 功能特性

### 🧮 个税计算器
- 支持工资薪金和年终奖两种计税方式
- 包含专项附加扣除计算
- 支持五险一金计算
- 自动计算应纳税所得额和应纳税额

### 🏠 房贷计算器
- 支持三种贷款类型：
  - 商业贷款
  - 公积金贷款
  - 组合贷款
- 支持两种还款方式：
  - 等额本息
  - 等额本金
- **还款计划明细功能**：
  - 弹出层展示详细还款计划
  - 支持分页浏览（10/20/50/100条/页）
  - 组合贷款支持分别查看合计、商业贷款、公积金贷款明细
  - 包含期数、月供、本金、利息、剩余本金等详细信息
- 计算结果包含：贷款总额、月供、还款总额、支付利息、还款月数

### 💱 汇率转换
- 支持全球多种主要货币
- 实时汇率转换
- 可搜索货币列表
- 支持双向转换

### 🔢 大写数字转换
- 将阿拉伯数字转换为中文大写数字
- 适用于财务、会计、合同、发票等正式场合
- 自动处理零的正确使用
- 支持整数和小数转换

### 📐 单位转换
支持多种物理量的单位转换：
- **长度转换**：毫米、厘米、米、千米、英寸、英尺、码、英里
- **面积转换**：平方米、平方厘米、平方千米、英亩、公顷、平方英尺、平方英寸
- **体积转换**：毫升、升、立方米、加仑、品脱、夸脱
- **重量转换**：克、千克、吨、磅、盎司、英石
- **温度转换**：摄氏度、华氏度、开尔文
- **压强转换**：帕斯卡、千帕、兆帕、巴、磅/平方英寸、标准大气压
- **时间转换**：毫秒、秒、分钟、小时、天、周、月、年
- **速度转换**：米/秒、千米/小时、英里/小时、节
- **功率转换**：瓦特、千瓦、马力、公制马力

### 💻 程序员工具
- **进制转换**：支持二进制、八进制、十进制、十六进制之间的相互转换
- **数据转换**：支持字节、千字节、兆字节、吉字节、太字节、拍字节的转换

### 📊 BMI计算器
- 根据身高体重计算身体质量指数
- 提供健康评估建议
- 显示BMI标准范围

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **部署平台**：Cloudflare Workers
- **开发工具**：Wrangler CLI

## 安装与运行

### 前置要求

- Node.js (v16 或更高版本)
- npm 或 yarn
- Wrangler CLI

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

访问 `http://localhost:8788` 查看应用。

### 部署到 Cloudflare Workers

```bash
npm run deploy
```

## 使用说明

### 个税计算器
1. 输入税前工资
2. 选择社保公积金缴纳方式（可选）
3. 输入专项附加扣除金额（可选）
4. 选择计税方式（工资薪金或年终奖）
5. 点击"计算"按钮获取结果

### 房贷计算器
1. 选择贷款类型（商业贷款、公积金贷款或组合贷款）
2. 输入贷款金额和年利率
3. 选择贷款年限和还款方式
4. 点击"计算"按钮获取计算结果
5. 点击"还款计划明细"按钮查看详细还款计划（仅在计算完成后可用）
6. 在弹出层中使用分页控件浏览还款计划

### 汇率转换
1. 输入要转换的金额
2. 选择原始货币
3. 选择目标货币
4. 点击"转换"按钮获取结果

### 大写数字转换
1. 输入要转换的数字
2. 点击"转换为大写"按钮获取结果

### 单位转换
1. 选择要转换的物理量类别
2. 输入数值
3. 选择原始单位和目标单位
4. 点击"转换"按钮获取结果

### 程序员工具
1. 选择工具类别（进制转换或数据转换）
2. 输入要转换的数值
3. 选择转换的源格式和目标格式
4. 点击"转换"按钮获取结果

### BMI计算器
1. 输入身高（厘米）
2. 输入体重（千克）
3. 点击"计算BMI"按钮获取结果

## 特色功能

### 房贷计算器还款计划明细
- **弹出层展示**：还款计划以弹出层形式展示，不占用主页面空间
- **分页浏览**：支持分页浏览，每页可显示10、20、50或100条记录
- **组合贷款支持**：对于组合贷款，可以分别查看合计明细、商业贷款明细和公积金贷款明细
- **固定分页控件**：分页功能固定在弹出层底部，不会因数据量变化而移动
- **完整分页功能**：支持首页、上一页、下一页、末页、页码跳转和分页大小切换
